# Security Policy

## Supported Versions

The latest published version of `react-motion-modal` is supported for security fixes.

## Reporting a Vulnerability

Please do not open a public GitHub issue for suspected vulnerabilities.

Report security concerns through one of these channels:

- GitHub Security Advisories for this repository
- Email the maintainer if a private contact channel is available on the project profile

Include the affected version, impact, reproduction steps, and any suggested mitigation if you have one.

## Release Integrity

This package is intended to be published through GitHub Actions using npm provenance.
Manual local publishing should be avoided for official releases.
